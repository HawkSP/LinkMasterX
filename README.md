# Project Title
LinkMasterX (SMLK-Pro)

## Description
This project provides a user-friendly way to move files or folders and then create symbolic links (symlinks) in their original location. It includes features for viewing, reverting, and recording symlink operations.

## Features
- Move and link files/folders with a single click.
- Keep track of your symlink history.
- Revert any past symlink creation.
- Simple watermark and optional custom application icon.

## Installation
1. Clone this repository.
2. (Optional) Create and activate a virtual environment.
3. Install dependencies:
```
pip install -r requirements.txt
```
4. Run the main application: ```python main.py```

## Usage
1. Launch the program.
2. Select a source file/folder.
3. Choose a destination folder.
4. Click "Move & Link" to move the item and create a symlink.
5. Open "History" to revert any symlink if needed.
